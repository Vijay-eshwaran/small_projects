Small Projects Portfolio

by Vijay(heisenberg)

Here you can find all my small projects. I've built them during my learning process, each one is a new challenge and I've tried to learn something new with each one, like a new library, a new framework, a new technique, etc.

Rock Paper Scissors Game

A simple game of rock paper scissors, where you can challenge the computer! Select your move (rock, paper, or scissors), and the computer will make its choice. The game tracks your score and provides instant feedback on each round's outcome.

Todo-list

A simple todo-list, where you can add your tasks and mark them as completed! You can also edit or delete your tasks.

Amazon

A simple Amazon clone that demonstrates a dynamic shopping experience. Features a product catalog with images, prices, ratings and reviews. Users can add items to their cart, adjust quantities, and see real-time cart updates. Built with HTML, CSS and JavaScript to practice e-commerce functionality.

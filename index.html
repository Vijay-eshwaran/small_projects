Small Projects Portfolio

by Vijay(heisenberg)

Here you can find all my small projects. I've built them during my learning process, each one is a new challenge and I've tried to learn something new with each one, like a new library, a new framework, a new technique, etc.

Rock Paper Scissors Game

A classic Rock Paper Scissors game where you compete against the computer! Key Features: • Simple one-click controls to make your move • Real-time score tracking and results • Automatic computer moves after each player turn • Mobile-friendly responsive design • Clean and intuitive interface Built using HTML, CSS and JavaScript to create a smooth and engaging gaming experience.

Todo-list

A simple and intuitive todo-list application where you can manage your tasks! Key Features: • Add, edit and delete tasks easily • Mark tasks as complete with visual feedback • Set due dates for your tasks • Tasks persist between sessions using local storage • Mobile-friendly responsive design • Clean and user-friendly interface Built using HTML, CSS and JavaScript to create a practical task management tool.

Amazon-clone

A clone of Amazon's shopping interface! Key Features: • Product catalog with images, prices and details • Functional shopping cart to add/remove items • Real-time cart total updates • Search functionality to find products • Mobile-friendly responsive design • Clean and intuitive interface Built using HTML, CSS and JavaScript to create an interactive e-commerce experience.
